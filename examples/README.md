# Examples

This folder will hold **synthetic** PFMEA and Control Plan files used by the quickstart and tests.

Planned (added with the vertical slice, see [../docs/ROADMAP.md](../docs/ROADMAP.md)):

- `pfmea.xlsx` — a small synthetic PFMEA following the recommended template.
- `control-plan.xlsx` — a matching Control Plan with **at least one deliberately seeded gap**
  (e.g. a high-severity failure mode with no corresponding control), so the report demonstrably
  surfaces a real finding.
- `report.md` — a sample generated report.

All example data is **synthetic** — no real or confidential customer content is ever committed.
